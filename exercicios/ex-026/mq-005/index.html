Testando Media Queris

1. Media Query

Definição: É uma funcionalidade do CSS que permite aplicar estilos específicos com base em características do dispositivo (como largura, altura, resolução, etc.).

Sintaxe básica: 
        
            css
            @media (condição) {
        estilos aqui 
            }
            
        Exemplo:
            css
            @media (max-width: 768px) {
            body {
                background-color: lightblue;
            }
            }

Aqui, o fundo do corpo mudará para azul claro em dispositivos com largura de até 768px.

2. Media Types

O que são:

Descrevem o tipo de dispositivo ou mídia ao qual o estilo será aplicado.

Tipos comuns:

- all: Padrão, aplica a todos os dispositivos.

- screen: Para telas de computadores, tablets, smartphones, etc.

- print: Para impressões em papel.

- speech: Para leitores de tela.

Exemplo: 
        
            css
                @media screen and (min-width: 1024px) {
            body {
                font-size: 18px;
            }

            }

Este estilo será aplicado somente para telas e quando a largura for de pelo menos 1024px.

3. Mobile First

Definição: E

stratégia de design onde o foco inicial é para dispositivos móveis, expandindo para telas maiores com media queries. Por que usar?: A maioria dos acessos à web vem de dispositivos móveis. Essa abordagem ajuda a criar sites otimizados desde o início.

Prática comum:

- Escreva os estilos base (mobile).

- Use media queries para telas maiores.

Exemplo: 
    
        css
        body {
            font-size: 14px; Estilo base (mobile) 
        }
        
        @media (min-width: 768px) {
            body {
            font-size: 16px; Estilo para tablets e desktops
            }
        }

4. Media Features

O que são:

Propriedades específicas que descrevem características do dispositivo.

Exemplos de features:

- width / height: Largura ou altura da viewport.

- max-width / min-width: Largura máxima ou mínima.

- orientation: Orientação da tela (portrait ou landscape).

- resolution: Resolução da tela (dpi, dpcm, etc.).

Exemplo: 
        
            css
            @media (orientation: landscape) {
            body {
                background-color: lightgreen;
            }
            }

O fundo será verde claro em dispositivos em modo paisagem.

5. Device Breakpoints

O que são:

Pontos de "quebra" onde o design do site muda para se adaptar melhor à largura do dispositivo.

Breakpoints mais usados:

- Mobile: até 480px

- Tablets: 481px a 768px

- Laptops pequenos: 769px a 1024px

- Desktops: 1025px ou mais

Exemplo prático:

        
            css
            Mobile 
            @media (max-width: 480px) {
            body {
                font-size: 12px;
            }
            }
            Tablets 
            @media (min-width: 481px) and (max-width: 768px) {
            body {
                font-size: 14px;
            }
            }
            Desktops 
            @media (min-width: 1025px) {
            body {
                font-size: 18px;
            }
            }

Resumo Prático

1. Use media queries para adaptar o design.

2. Conheça os media types para alvos específicos como screen e print.

3. Aplique o mobile first para designs responsivos.

4. Trabalhe com media features como max-width e orientation.

5. Planeje seus breakpoints para diferentes dispositivos.
